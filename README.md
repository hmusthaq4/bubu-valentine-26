# Bubu Valentine (Vite + React + TypeScript)

A cute pale-pink Valentine app with:
- Scrapbook photo collage background
- Runaway "No" button
- Confetti on "Yes"
- After "Yes": full-screen collage takeover
- Sticker PNGs around the card (replaceable)

## Install
```bash
npm install
npm run dev
```

## Replace photos
Put your photos in `src/assets/photos/` (jpg/png/webp). Remove the sample ones if you want.

## Replace stickers
Replace files in `src/assets/stickers/`:
- heart.png
- cloud.png
- star.png
